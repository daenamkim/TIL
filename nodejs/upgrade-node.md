# Upgrade Node

Install `n` to upgrade node.

```sh
$ sudo npm install -g n
```

Check versions.

```sh
$ n --stable
$ n --latest
```

Upgrade node.

```sh
$ sudo n latest
```

After cloing and opening a new console updated node will be appiled.

```sh
$ node -v
v10.2.1
```
